blake_trunc_add_model and skein_trunc_add_model are C programs that replace addition with truncated-carry addition in the Blake and Skein hash algorithms.

The library source files are:

  • blake_trunc_add_model.c, based on blake256_light.c from http://www.131002.net/blake/
  • skein_trunc_add_model.c

To compile these, we use 
 gcc -o blake_trunc_add_model -O3 blake_trunc_add_model.c
and 
 gcc -o skein_trunc_add_model -O3 skein_trunc_add_model.c
respectively.
