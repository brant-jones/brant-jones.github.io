liberiksson is a C++ library for performing basic operations on Coxeter elements. It includes a client program which classifies Deodhar elements of finite Coxeter groups by embedded factor containment.

The library source files are:

  • CoxeterSystem.h
  • CoxeterSystem.cpp
  • CoxeterElement.h
  • CoxeterElement.cpp

The Makefile compiles these into a shared library (liberiksson.so on unix, liberiksson.dll on cygwin).
The client file is classify.cpp.
See the README.txt for more information and sample output.
